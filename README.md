*Vite* - (Faster than create-react-app.)

*Initialization*

npm create vite@latest ./ -- --template react

*Run App*

npm run dev


*Additional Dependencies*

Tailwind CSS:

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
